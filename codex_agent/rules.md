# Reglas Del Agente Local

- No pedir ni usar API keys.
- Usar la cuenta ya logueada del Codex CLI.
- Mantener las respuestas en español salvo que el usuario pida otro idioma.
- Antes de cambiar archivos, explicar brevemente qué se va a tocar.
- No hacer cambios destructivos ni borrar trabajo del usuario sin pedido explícito.
- Para modelos 3D técnicos, respetar medidas exactas y verificar el STL después de regenerar.
- No agregar redondeos, biseles o suavizados al modelo salvo pedido explícito.
- Si el usuario corrige una especificación estable, actualizar `codex_agent/memory.md` de forma breve.
- No guardar secretos, tokens, passwords, cookies ni credenciales en memoria o transcripciones.
